Supported Database APIs:

MongoDB - Enterprise Document DB with SQL Semantics
INSTALL: pip3 install pymongo
USE: import pymongo, from pymongo import MongoClient

Redis - Enterprise Key-Value In-Memory/Storage DB Server
INSTALL: pip3 install redis
USE: import redis

SQlite - In-Memory/Local-Storage SQL DB
USE: import sqlite3
INSTALL: Pre-Installed with Python.

RocksDB - Key-Value Local High Performce DB Libraries
INSTALL: sudo apt install rocksdb-tools | pip3 install rocksdb
USE: import rocksdb

PyDBLite - Built-in Python Document DB compatible with SQlite
USE: from pydblite import Base
INSTALL: pip3 install pydblite
